# Notifications

Notifications help you respond to collaboration events without searching through every project manually.

## Where notifications appear

Open the notifications area from the app navigation. You may see notifications for:

- Project invitations.
- Membership requests.
- Updates about request decisions.
- Language link suggestions.
- Project language suggestions.

Notifications sync when your device is online. If you have been offline, open the app on a stable connection and allow time for new notifications to appear.

## Project invitations

When a project owner invites you:

1. Open **Notifications**.
2. Find the project invitation.
3. Review the project name and role.
4. Accept if you want to join.
5. Dismiss or decline if the invitation is not relevant.

Invitations are tied to the email address used for your LangQuest account. If you do not see an expected invite, ask the project owner to confirm the address they used.

## Membership requests

If you request access to a public or private project, the project owner can approve or reject the request.

Owners may receive notifications when someone asks to join. Review the request before approving, especially for private projects.

## Language suggestions

LangQuest may suggest linking a custom language entry to a known language code or catalog entry. This helps keep projects discoverable and consistent.

When reviewing a language suggestion:

1. Check whether the suggested language matches the community and project.
2. Accept the suggestion if it is correct.
3. Dismiss it if the match is wrong or uncertain.

If you are not sure, ask a project leader or language specialist before accepting.

## Project language suggestions

Some notifications may suggest that a project's language information should be updated. These suggestions can help when a project name, target language, or catalog link does not match cleanly.

Accept suggestions only when they accurately describe the project. Incorrect language metadata can make projects harder to find and manage.

## Email notifications

Some collaboration events may also send email, such as project invites or membership requests. Email can be delayed or filtered by spam settings.

If someone does not receive an expected email:

1. Confirm the recipient's LangQuest account email.
2. Ask them to check spam or junk folders.
3. Have them open the in-app notifications area.
4. If needed, cancel and resend the invite.

## Tips for project owners

- Invite people using the email they used to register.
- Explain the project before inviting a large group.
- Review membership requests before approving access to private projects.
- Keep project language metadata accurate.
- Use owner roles carefully; owners can invite and promote other members.
